Filtrs tiek izmantots arī atskaišu \nsagatavošanā un atsevišķu dokumentu satura aizpildīšanā.</p>\n<p>Datu atlases kritērijus var saglabāt (<img src="Skin/help/img.aspx?pid=24938" border="0">).
