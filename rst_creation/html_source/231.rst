ssw_id = 33skf_id = 231Cenošanas dokumenti;<P>Cenošanas dokumentu žurnālā&nbsp;tiek saglabāti visi izveidotie&nbsp;<A href="/help/?ssw=33&amp;skf_id=374">Cenošanas dokumenti</A>.&nbsp;&nbsp;</P>\n<P>Lai atlasītu dokumentus Cenošanas dokumentu žurnālā, jāizmanto datu atlases filtrs. Dati uz ekrāna tiek parādīti atbilstoši norādītajiem datu atlases kritērijiem filtra ekrāna formā. Filtru var uzlikt, lai ierobežotu kādu datu parādīšanu uz ekrāna, iegūstot nepieciešamās izdrukas, vai arī ātrāk sameklētu nepieciešamos datus. Filtrs tiek izmantots arī atskaišu sagatavošanā un atsevišķu dokumentu satura aizpildīšanā.</P>\n<P>Datu atlases kritērijus var saglabāt (<IMG src="/help/Skin/help/img.aspx?pid=24938" border=0>). Izvēloties izveidoto filtra sagatavi (<IMG src="/help/Skin/help/img.aspx?pid=24943" border=0>), filtrs tiek automatizēti aizpildīts no filtra sagataves.</P>
