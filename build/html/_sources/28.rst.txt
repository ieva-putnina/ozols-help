.. 28 =================Konfigurācija================= 




Izmantojot Algu moduļa Konfigurācijas darba režīmus, iespējams
nodefinēt LR atbilstošās nodokļu likmes:


+ `Ienākuma nodokļu metodes`_
+ `Sociālo nodokļu metodes`_


Izveidot algu konfigurāciju, definējot atbilstošos algu kodus:


+ `Algu konfigurācijas sistēmas mainīgie`_
+ `Ieejošo kodu konfigurācija`_
+ `Ieejošo kodu veidi`_
+ `Rezultējošo kodu konfigurācija`_
+ `Rīkojumu veidu ieejošo datu konfigurācija`_
+ `Sistēmas kodi`_


Izveidot Kolonnu virsrakstu konfigurācijas `Algu atskaitēm`_:


+ `Kolonnu virsraksti`_


Kā arī pievienot jaunus `Algu periodus`_un nodefinēt `Individuālos`_un
`Kopīgos`_konfigurācijas iestādījumus.
