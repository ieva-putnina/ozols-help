Sistēmas lietošanas instrukcija¶

Sadaļā pieejamas sistēmas Ozols lietošanas instrukcijas, kas palīdzēs labāk izprasts programmas funkcionalitātes iespējas un darbības principus.

Lietotājiem

  • Sistēmas lietošanas principi
    • Darba režīmu izkārtojums un pieejamās komandas
    • Darba režīmu pārskats
    • Datora iestādījumu konfigurācijas parametri
      • Iestādījumu konfigurācija strādājot ar OS Windows Vista
  • Administratora instrukcijas
    • Datu bāzes atjaunošana no BackUp faila, izmantojot SQL Enterprise Manager
    • Datu bāzes lietotāju paroles konfigurācija
    • Datu bāzes lietotāju tiesības darba režīmiem
      • Lietotāju lomu funkcijas
      • Fiksētās datu bāzes lomas
      • Ieteicamās lietotāja definētās datu bāzes lomas
